# lungPWSimulation
    肺部超声PW仿真。
    仿肺部运动，仿超声探测的信号。
### OneDimension
    一维的肺部仿体，使用卷积的方式仿真超声回波信号。
### TwoDimension
    二维的肺部仿体，先使用了一张图片作为仿体的框架。
### Field_II_Sim
    将一维仿体按照时间顺序排列，后使用仿真工具Field II计算超声回波。
### lungSound
    通过超声测量肺部声音。
### convolutionSim.m
    最初的一维仿体卷积仿真。
### antroll.m
    仿真蚂蚁爬。
### phaseDemodulation.m
    信号相位解调的小例子。
